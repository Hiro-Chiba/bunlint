# テキスト統計・句読点変換ユーティリティ

このリポジトリでは、テキストの統計情報を算出する処理と、句読点スタイルを相互変換する処理を提供しています。Next.js アプリ本体はまだ未着手ですが、ドメインロジックとテストの整備を先行して進めています。

## 提供機能

- **テキスト統計 (`src/lib/text.ts`)**
  - 文字数（結合文字を含む）、単語数、文数を算出。
  - 改行や連続した終端記号を含むテキストでも安定して集計。
- **句読点変換 (`src/lib/punctuation.ts`)**
  - 和文スタイル（、。）と学術スタイル（，．）を相互に変換。
  - テキスト内の多数派から採用する句読点スタイルを簡易判定。

## セットアップとテスト

```bash
bun install
bun test
```

テストは `tests/` 配下に用意しており、ユースケースの追加に合わせて拡充しています。
